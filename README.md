# Streaming Overlay System

一個專為直播主設計的覆蓋層管理系統

## 📋 Claude Code 使用規範

使用 Claude Code 開發此專案時,請遵循以下規則:

### 💬 輸出效率
- **輸出盡量簡潔,減少 token 使用**
- 成功訊息簡短確認即可
- 避免重複解釋

### ⚡ 程式修改
- **直接執行,無需詢問**
- 發現問題直接修復
- 完成後簡短回報

### 認證

- 可能需要生產環境變數

### 錯誤處理
